# 完美B1

主要展示历史完美B1案例,从中学习和获取分析策略

B1 策略主要是通过量价关系,分析主力建仓-洗盘-拉升-出货(吸拉派落)的整个过程,从而找到合适的买入点

这里罗列了一些历史B1案例的情况

量价关系中特别要留意买入倍量柱的位置,倍量柱子附近的价格大概率是主力建仓的平均成本价

## 完美案例

A

- 688799 华纳药厂 2025-04-16 到 2025-04-24 建仓波带量起来; 2025-04-25 到 2025-05-12 缩量回调洗盘一直到黄线强支撑,出现B1(这是个SB1),小阴线 -0.92%; 后续16个交易日 涨幅80%以上;
- 600366 宁波韵升 2025-06-04 到 2025-06-17 第一个建仓波; 2025-06-18 到 2025-06-27 缩量回调多个B1; 2025-06-30 到 2025-08-06 拉升后回调出现B1+红砖,小阳线 1.79%; 后续9个交易日 涨幅50%以上;

- 600601 方正科技 2025-06-03 到 2025-06-11 第一个建仓波,带量站上黄线; 2025-06-12 到2025-06-24 回调到黄线附近,缩量,多个B1, 2025-06-24 还是B1+红砖的共振, 小阳线 1.7%; 之后从 2025-06-25  到 2025-07-=08 第二波拉升, 涨幅 20%以上; 2025-07-09 到 2025-07-23 再次缩量回调到白线附近, 2025-07-23 白线以下B1 小阴线 -1.49%; 后续65个交易日 涨幅 150%以上

### 选股快速匹配流程

> 当实盘扫描到B1信号时，按以下流程快速匹配最接近的模式

**第一步：看回调缩量评分**
- <20% → 优先匹配**光电股份模式**
- 20%~28% → 进入第二步
- 28%~35% → 优先匹配**华纳药厂/国轩高科/微芯生物模式**
- >35% → 大概率是**赢时胜模式**（短期B1，涨幅有限）

**第二步：看白线与黄线关系**
- 白线全程不死叉黄线 → **澄天伟业模式**或**国轩高科模式**
- 跌破白线但不破黄线 → **微芯生物模式**
- 跌破黄线 → **野马电池模式**或**新瀚新材模式**

**第三步：看结构特征**
- 前期翻倍(>100%) → **昂利康模式**
- 三波以上N型结构 → **宁波韵升模式**
- 双波建仓+第二波量价齐升 → **国轩高科模式**
- 超长建仓+多次倍量柱 → **新瀚新材模式**
- 同一股票第二次B1 → **方正科技模式**（第二次更可靠）

**第四步：确认信号强度**
- B1+红砖共振 → 宁波韵升/野马电池/光电股份（更强）
- B1后1~3天出现倍量柱 → 澄天伟业/微芯生物（确认买入点）

### 倍量柱与建仓成本

量价关系中特别要留意倍量柱的位置。倍量柱子附近的价格大概率是主力建仓的平均成本价。

从案例中可以观察到：
- **倍量柱出现在建仓波中**（华纳药厂、微芯生物）：标记主力成本区间，后续回调不跌破此价格说明主力仍在
- **倍量柱出现在B1后**（澄天伟业B2、微芯生物6/23）：确认信号，B1后1~3天出现倍量柱是更强的买入确认
- **倍量柱出现在拉升中**（新瀚新材7/14和7/28）：多次倍量柱说明主力反复建仓加仓，建仓期越长后续爆发力越强
